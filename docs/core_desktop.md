# Core — Tauri Desktop App

## このアプリが解決すること

ローカルの `.md` ファイル群を、キーボードだけで高速に操作・編集できる環境を作る。
Obsidian の代替だが、vim 操作と neovim ライクな UX を中心に設計する。

---

## ユーザーストーリー

**ファイルを探す**
- フォルダツリーを `j/k/h/l` で移動してファイルを開きたい
- `cmd+O` でファイル名を fuzzy 検索して即座に開きたい
- `cmd+shift+F` でファイルの中身を横断検索したい
- `#タグ` で絞り込んで関連ノートを探したい

**書く・編集する**
- vim keybindings で編集したい（normal / insert / visual モード）
- 書きながら右側にレンダリング済みのプレビューを見たい
- `[[ファイル名]]` と書くとリンクになり、そのファイルに飛べる
- 自動保存されるので保存を意識したくない

**つながりを把握する**
- 今開いているノートを参照している他のノート（バックリンク）を見たい
- `[[wikilink]]` の先がどこかをその場で確認したい

**同期する**
- 編集が一段落したら git commit & push をアプリ内からやりたい
- iPhone 側で書いた変更を pull して取り込みたい
- オフラインでも編集できる。オンラインになったら同期する

---

## 優先順位

1. vim 操作感（これが崩れるとアプリの意味がない）
2. ファイルを開くまでの速度（fuzzy 検索の精度・レスポンス）
3. プレビューの見やすさ
4. バックリンク・タグ
5. git 連携
