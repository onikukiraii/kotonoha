# Core — Web App（自宅サーバー）

## このアプリが解決すること

iPhone から自宅サーバー経由で `.md` ファイル群にアクセスし、閲覧と軽量編集ができる環境を作る。
Mac アプリと同じファイルを見ている、という一体感が重要。

---

## ユーザーストーリー

**ファイルを探す・読む**
- フォルダツリーをタップして目的のノートを開きたい
- 検索ボックスでファイル名を fuzzy 検索したい
- Markdown がきれいにレンダリングされた状態で読みたい
- `[[wikilink]]` をタップしてリンク先のノートに飛びたい

**書く・編集する**
- 短いメモを書き足したい（数行〜数十行程度）
- 新規ファイルを作ってその場で書き始めたい
- 保存は自動でやってほしい（ボタンを押したくない）
- 編集した内容が git commit されて Mac 側と同期される

**同期する**
- アプリを開いたとき、Mac 側の最新状態が反映されている
- iPhone で書いた内容が Mac で pull したときに入ってくる

---

## Mac アプリとの違い

| | Mac アプリ | Web アプリ |
|---|---|---|
| 操作 | キーボード・vim | タッチ |
| 編集規模 | がっつり書く | 短いメモ・追記 |
| オフライン | 必須 | 不要 |
| vim mode | あり | なし |
| git 操作 | 手動で行う | 自動 commit |

---

## 優先順位

1. ファイルを開いて読める（閲覧体験）
2. 短い編集・新規作成ができる
3. Mac 側との同期が壊れない
4. 検索
5. バックリンク・タグ
